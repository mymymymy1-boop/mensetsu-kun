# E2Eシナリオ — メンセツくん（Playwright・音声はモック注入）

各シナリオ7項目: ID/目的/前提/手順/期待/異常時/トレース。音声API(TTS/録音/STT)はテスト用モックをwindowに注入し決定論化。

## 正常系
### E2E-N1: 共通モードで1セッション完走（子供フロー）
- 目的: 出題→回答→FB→次へ→結果 が成立(FR-SEL/AUD/REC/SCORE/FB)
- 前提: モックTTS/録音を注入(録音は2秒相当)
- 手順: ホーム→「きょうつう」→「めんせつ」→出題画面でマイクタップ→停止→FB表示→規定問数繰返し→結果へ
- 期待: 各問でキャラのほめ言葉/星が出る・数値スコアは画面に無い・最後にセッション結果(子供向け)
- 異常時: 録音モックnull→「もういちど」案内で継続しクラッシュしない
- トレース: FR-SEL-001/002,FR-AUD-001,FR-REC-001,FR-SCORE-001,FR-FB-001

### E2E-N2: 慶応幼稚舎を選ぶと制作お尋ね中心＋注記表示
- 目的: 校別傾向と注記(FR-DATA-002/FR-SYS-001)
- 手順: ホーム→「けいおうようちしゃ」→出題
- 期待: 「面接なし・制作お尋ね中心」注記、artworkカテゴリ質問が出る、公式確認の注意が見える
- 異常時: データ不足時はcommonで補完
- トレース: FR-DATA-002,FR-SYS-001,FR-SEL-001

### E2E-N3: 保護者ゲート→結果確認→採点補正
- 目的: 大人領域保護と補正(FR-GATE/RPT/SCORE-002)
- 前提: 1セッション完了済み
- 手順: 右上ゲート入口→計算正解→結果一覧→ある問のスコアを補正・メモ入力→保存
- 期待: 結果に出典/信頼度/録音再生/合計、補正が反映、子供画面に戻ると数値非表示
- 異常時: ゲート不正解で結果に入れない
- トレース: FR-GATE-001,FR-RPT-001,FR-SCORE-002

## 異常系
### E2E-E1: STT非対応ブラウザでrecord-onlyフォールバック
- 目的: 非対応でも継続(NFR-COMPAT/REL,FR-STT-001)
- 前提: stt.supported()=falseのモック
- 手順: 設定でSTTを有効化しようとする→出題→回答
- 期待: STTは呼ばれず録音のみで採点(keywordHit=unknown)、警告表示、クラッシュなし
- 異常時: -（本シナリオ自体が異常系）
- トレース: FR-STT-001,NFR-COMPAT-001,NFR-REL-001

### E2E-E2: マイク不許可で口頭練習モード継続
- 目的: getUserMedia拒否時の継続(FR-REC-001例外)
- 前提: 録音supported=false/許可拒否モック
- 手順: 出題→マイクタップ
- 期待: 「読み上げ＋口頭練習(保護者採点)」案内、出題と次へは継続
- 異常時: -
- トレース: FR-REC-001,NFR-REL-001

## 境界系
### E2E-B1: 無回答(沈黙)でも否定されず励まし
- 目的: 無回答の教育的扱い(FR-FB-001,FR-SCORE-001境界)
- 前提: 録音モックがdurationMs=0/null
- 手順: 出題→何もせず停止→FB
- 期待: 星0でも「もういちど やってみよう」、否定語なし、次へ進める
- 異常時: -
- トレース: FR-SCORE-001,FR-FB-001,C-EDU-002
