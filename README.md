# メンセツくん — 小学校受験 子供向け 音声面接対策アプリ

年長児（5〜6歳）が、家庭で「面接・行動観察の受け答え」を**音声で練習**できるWebアプリです。
音声で質問が読み上げられ → お子さまが声で答え → 録音（任意で自動聞き取り）→ 採点 → 保護者が振り返れます。

対応校: **慶応横浜初等部 / 慶応幼稚舎 / 桐蔭学園小学校 / 森村学園初等部** ＋ 共通 ＋ 行動観察の声かけ。

---

## つかいかた（開発・実行）

```bash
npm install
npm run dev       # 開発サーバー（http://localhost:5173）
npm run build     # 本番ビルド（dist/ に静的ファイル生成・サーバー不要）
npm run preview   # ビルド結果をプレビュー（http://localhost:4173）
npm test          # 単体・結合テスト
npm run coverage  # カバレッジ
npm run e2e       # E2E（要: npx playwright install chromium）
```

`npm run build` で出力される `dist/` を、任意の静的ホスティング（GitHub Pages 等）やローカルで開くだけで動きます。**外部サーバー・APIキー・アカウント・課金は不要**です。

## 推奨環境
- **Chrome または Safari の最新版**を推奨（音声読み上げ・録音が安定）。
- 自動の聞き取り（STT）は Chrome / Safari のみ対応。非対応ブラウザでは自動で「録音のみ」になります。

## 大切なこと（保護者の方へ）

### プライバシー
- お子さまの声は**既定では端末の中だけ**で扱い、**外部に送信しません・保存しません**。
- 「自動の聞き取り」を有効にした場合のみ、声がブラウザの音声認識サービス（Google等）に送信されます。これは設定（保護者ゲートの奥）で**明示的に同意したときだけ**有効になります。既定はオフです。

### 採点について
- 5〜6歳の発話は自動音声認識が苦手な年齢帯です（誤認識が起きやすい）。そのため**自動採点はあくまで補助**で、保護者が録音を聞いて手動で補正できます。
- お子さまには**点数（数字）を見せず**、花丸・星・ほめ言葉で表現します。できなくても否定せず励まします。

### 受験情報について
- 学校は面接の質問内容を公式に公開していません。本アプリの質問は、各校公式（面接の有無）＋大手幼児教室・合格体験談に基づく**参考**で、各質問に**信頼度（A〜D）と出典**を表示しています。
- **面接の有無・形式は年度で変わります。最新は必ず各校公式の募集要項でご確認ください。**
  - 慶応2校は面接がなく、絵画・制作中の「お尋ね」が中心です。
  - 桐蔭は親子面接＋個別口頭試問、森村は保護者面接＋子供のグループ面接が中心です（調査時点）。

詳しい調査根拠は `research/`（research_v1.md / research_v2.md / question_bank.md）を参照してください。
